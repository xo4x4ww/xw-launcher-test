# XW Launcher 🚀

**Неофициальный лаунчер для Minecraft: Java Edition**

> 🧠 Почти полностью создан ИИ. Кроме промптов, иконок, обложек и этого README — их делаю я 😎

Лаунчер использует официальное API Mojang. Рекомендуется наличие **лицензионного аккаунта** Minecraft.

---

## ✨ Возможности

- ✅ Загрузка официальных версий Minecraft через Mojang API
- ✅ Установка и запуск игры с выбором версии
- ✅ Управление оффлайн-профилями (добавление/удаление/переключение)
- ✅ Просмотр установленных модов с автоматическим извлечением иконок
- ✅ Настройка выделения оперативной памяти (RAM)
- ✅ Автоматическое закрытие лаунчера при старте игры
- ✅ Новости версий Minecraft
- ✅ Полностью рабочий GUI на Tkinter

---

## 🖥️ Системные требования

- Windows 10 / 11
- Установленная Java 17 или новее
- Интернет для загрузки версий

---

## 🚀 Запуск для разработки

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/xw-launcher.git
cd xw-launcher/code

# Создать и активировать виртуальное окружение
python -m venv .venv
.venv\Scripts\activate

# Установить зависимости
pip install minecraft-launcher-lib

# Запустить лаунчер
python run.py
```

## 🛠️ Сборка .exe файла

```
pip install pyinstaller
python build.py
```

Результат: `dist/XWLauncher.exe`
